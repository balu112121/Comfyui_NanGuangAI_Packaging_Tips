# 南光AI包装提示词插件

## 📦 简介

南光AI包装提示词是一个专为ComfyUI设计的智能包装设计提示词生成插件。通过输入产品信息和设计需求，自动生成专业的包装设计AI绘画提示词。

### ✨ 主要特性

- 🎯 **10个专项输入字段**：产品、品牌名、尺寸规格、净含量、包装样式、设计风格、设计元素、主题配色、印刷材质、印刷工艺
- 🤖 **智能提示词生成**：自动构建专业的包装设计AI绘画提示词
- 🌍 **多语言支持**：支持中文和英文输出
- 🎨 **多种输出格式**：详细完整版、简洁版、关键词版
- 💡 **风格强调功能**：标准、高端感、生态环保、简约现代、传统韵味
- 🔧 **文本长度控制**：1000字符限制
- 📊 **调试信息输出**：便于问题排查

## 🚀 快速安装

1. 打开ComfyUI Manager

3. 点击安装
